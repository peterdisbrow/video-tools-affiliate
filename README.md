# Video Creator's Toolkit - Affiliate Landing Page

A simple, high-converting affiliate landing page for video production gear and software recommendations.

## Features

- 18+ curated products (cameras, audio, lighting, software, streaming, accessories)
- Real affiliate links to Amazon, B&H Photo, and other retailers
- Responsive design for mobile and desktop
- Product filtering by category
- Social proof section with testimonials
- FAQ accordion
- SEO optimized

## Deployment

Deployed on Vercel. Updates automatically when you push to GitHub.

## How to Add Products

Edit the `products` array in `app/page.js` to add or remove products.

## Affiliate Program Setup

1. **Amazon Associates**: https://affiliate-program.amazon.com/
2. **B&H Photo**: https://www.bhphotovideo.com/
3. Update the `link` field in products to use your affiliate IDs
